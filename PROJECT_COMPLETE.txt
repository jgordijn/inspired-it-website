💾 CREATING BLOG POSTS (Two Methods)
═══════════════════════════════════════════════════════════════════════════

METHOD 1: Use the Converter Tool (Easiest) ⭐
  1. Go to http://localhost:3000/admin/markdown-converter
  2. Edit YAML frontmatter (title, tags, description, etc.)
  3. Write your post in Markdown
  4. See live HTML preview
  5. Click "Download Blog Post File"
  6. Save to content/blog/ folder
  7. Run npm run build
  8. Blog post appears automatically!

METHOD 2: Manual File Creation
  1. Create file: content/blog/my-post.md
  2. Add this format:
     ---
     title: "Post Title"
     description: "Short description"
     date: "2024-10-16"
     author: "Your Name"
     tags: ["tag1", "tag2"]
     ---

     # Post Title

     Your Markdown content here...

  3. Save and run npm run build
